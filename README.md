# 验证码识别系统 (Captcha Recognition System)

基于深度学习的验证码自动识别系统，支持4-6位数字字母验证码。

## 特性

- 🎯 高准确率：基于CNN+LSTM架构，识别准确率90%+
- 🚀 易于使用：简洁的API接口，支持多种语言调用
- 🔧 可定制化：支持自定义训练数据和模型参数
- 📊 完整工具链：数据生成、模型训练、评估一体化
